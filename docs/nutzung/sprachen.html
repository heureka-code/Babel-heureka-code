Sprachen¶

Basisklasse¶

Für Sprachen wird eine abstrakte Klasse definiert

class Babel_heureka_code.BasisBabelSprachVersion¶

Basisklasse fuer die Sprachkonfigurationen

abstract wert_fuer_code(code: str) → str¶

Ermittelt den uebersetzten Text fuer den Code

Implementierung¶

Es gibt mehr als eine Standardimplementierung

Eine Sprache kann innerhalb des Codes über ein Dictionary definiert werden¶

class Babel_heureka_code.DictSprache(dic: Optional[dict] = None, **kwargs)¶

Erstellt eine Sprachkonfiguration im Code

wert_fuer_code(code: str) → str¶

Ermittelt den referenzierten Wert

Auch kann eine Sprache über eine Konfigurationsdatei definiert werden¶

class Babel_heureka_code.SprachConfig(datei, bereich: Optional[str] = None)¶

Verwaltet eine Sprachkonfiguration aus einer Datei

wert_fuer_code(code: str) → str¶

Ermittelt die passende Uebersetzung

Anwendung¶

>>> from Babel_heureka_code import BabelCode, BabelVerwalter, DictSprache
>>> verwalter = BabelVerwalter()
>>> code1: BabelCode = verwalter.code01
>>> code1
<BabelCode code01>
>>> de = DictSprache({"code01": "Erster code"})
>>> en = DictSprache(code01="First code")  # Eine Sprache kann auch über kwargs erstellt werden
>>> de.wert_fuer_code("code01")
'Erster code'
>>> en.wert_fuer_code("code01")
'First code'
>>> verwalter.setze_sprache(de)
>>> code1.wert
'Erster code'
>>> verwalter.setze_sprache(en)
>>> str(code1)
'First code'
